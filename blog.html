1.FONT-STYLE
now i know how to style the text on my website using CSS for eg.

I made this para italic without warping it in the "em" tag, instead i made use of the css property "font-style"
The "font-style" tag consists of 3 different value

  1. normal
  2. italic
  3. oblique

2.FONT-WEIGHT

just like the "font-style" "font-weight" also styles the text when CSS is used on it

the only difference between "font-style" and "font-weight" is that "font-weight" changes the the thickness of the letters
the "font-weight" tag consists of 3 values

  1. normal
  2. bold
  3. 900

3.FONT-FAMILY

"font-family" tag can be used to choose what type of font text you would like your users to see. There are 1000s of font families u can choose from but
These 3 are present on almost every pc by default

  1. serif
  2. sans-serif
  3. monospace

4.LINE-HEIGHT

"line-height" tag can be used to adjust the space between lines of text
many values are used in this tag for eg.

  1. normal
  2. precentage
  3. lenght(em,px,rem)
  4. number
